
# Imagicnation - Based on OpenAI's GPT3




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/imagicnation.git
```

Go to the project directory

```bash
  cd imagicnation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
